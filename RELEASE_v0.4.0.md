# My Azure Portal Extension v0.4.0

## Highlights

- Authentication experience refreshed for modern usage.
- Sign in from popup UI (`Sign in` button) instead of manual token copy/paste.
- Token lifecycle handling improved with automatic refresh behavior.
- Ongoing maintenance updates for current extension configuration.

## Why this release matters

This is a major milestone in the project lifecycle: a practical authentication refresh after long-term maintenance.

## Upgrade notes

- If you used old manual token operations in older builds, switch to popup-based sign-in.
- Configure Tenant ID and Client ID once, then sign in from the extension popup.
